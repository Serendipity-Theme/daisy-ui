# Serendipity for [Platform]


Learn how to use it here:

- https://daisyui.com/theme-generator/

Midnight

```
module.exports = {
  daisyui: {
    themes: [
      {
        mytheme: {
          primary: "#EE8679",
          secondary: "#F8D2C9",
          accent: "#5BA2D0",
          neutral: "#DEE0EF",
          "base-100": "#151726",
          info: "#94B8FF",
          success: "#33ddbe",
          warning: "#f6c33f",
          error: "#F87272",
        },
      },
    ],
  },
  plugins: [require("daisyui")],
  //...
};

```
